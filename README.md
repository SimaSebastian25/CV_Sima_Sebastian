
# Personal CV Website

This project is a personal CV/resume webpage built using **pure HTML and CSS**.
The goal of the project is to present a clean and professional resume layout that can also be easily exported as a **PDF (A4 format)**.

## Features

* Clean and structured CV layout
* Built using **semantic HTML**
* Styled with **modern CSS**
* Optimized for **printing and PDF export**
* Simple and lightweight (no frameworks or JavaScript)

## Technologies Used

* HTML5
* CSS3

## Preview

The CV includes sections such as:

* Contact information
* Skills
* Education
* Professional experience
* Projects
* Additional information

## Usage

1. Clone the repository

```bash
git clone https://github.com/SimaSebastian25/CV_Sima_Sebastian.git
```

2. Open `index.html` in your browser.

3. To export the CV as a PDF:

   * Press **Ctrl + P**
   * Choose **Save as PDF**
   * Set **Margins: None**
   * Enable **Background graphics**
